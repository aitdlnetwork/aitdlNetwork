# Deep Space Elegance - AITDL Network ## Product Overview **The Pitch:** A premium, futuristic SaaS marketing platform for an AI-first technology consultancy. It communicates enterprise-grade reliability through cutting-edge, bioluminescent design cues and seamless glassmorphic interfaces. **For:** Startup founders and enterprise CTOs seeking high-end AI integration, custom software, and scalable intelligent systems. **Device:** desktop **Design Direction:** Deep Space Elegance - vast dark indigo voids punctuated by soft, bioluminescent teal and emerald glows, floating glass panels, and striking geometric typography. **Inspired by:** Vercel, Linear, Palantir --- ## Screens - **Home View:** Immersive hero with floating value propositions and high-level service overview. - **Services View:** Detailed breakdown of technical capabilities in an asymmetrical, glowing grid. - **Portfolio View:** Cinematic showcase of three core demo projects with deep-dive overlays. - **Contact View:** High-conversion terminal-inspired contact form and direct WhatsApp integration. --- ## Key Flows **Lead Generation Flow:** Enterprise client exploring AI automation 1. User is on Home View -> sees glowing "Intelligent Systems" hero section 2. User clicks `Get Started` -> scrolls smoothly to Contact View 3. User fills out glassmorphic email form or clicks WhatsApp CTA -> success state glow triggers ---
Design System ## Color Palette - **Primary:** `#00F0FF` - Cyan/Teal glow for main CTAs, active states, text highlights - **Background:** `#070B14` - Deep Space Navy, absolute background - **Surface:** `#111827` - Base card color, often paired with `rgba(17, 24, 39, 0.6)` for glass effect - **Text:** `#F8FAFC` - Primary headings, high-emphasis text - **Muted:** `#8A96AA` - Body copy, secondary information, subtle borders - **Accent:** `#00FF9D` - Emerald bioluminescence for success states, secondary gradients ## Typography - **Headings:** Space Grotesk, 700, 32-64px - **Body:** Outfit, 400, 16-18px - **Small text:** Outfit, 300, 14px - **Buttons:** Space Grotesk, 600, 15px, 1px letter-spacing **Style notes:** - **Glassmorphism:** `backdrop-filter: blur(24px)` with `border: 1px solid rgba(255, 255, 255, 0.08)` - **Glows:** Absolute positioned radial gradients (`radial-gradient(circle, rgba(0,240,255,0.15) 0%, transparent 70%)`) behind cards - **Shadows:** `box-shadow: 0 8px 32px rgba(0, 0, 0, 0.4)` - **Radii:** Sharp, technical 12px for large panels, 6px for buttons ## Design Tokens ```css :root { --color-primary: #00F0FF; --color-background: #070B14; --color-surface: rgba(17, 24, 39, 0.6); --color-text: #F8FAFC; --color-muted: #8A96AA; --color-accent: #00FF9D; --font-heading: 'Space Grotesk', sans-serif; --font-body: 'Outfit', sans-serif; --radius-card: 12px; --radius-button: 6px; --spacing-section: 120px; } ```

Screen Specifications ### Home View **Purpose:** Hook high-value clients and establish immediate technical authority. **Layout:** Full viewport hero with centralized text, floating 3D/glass elements, followed by a continuous scrolling layout. **Key Elements:** - **Hero Headline:** 64px Space Grotesk, text-gradient (white to muted), "We Build Intelligent Systems That Scale Businesses" - **Hero Subtext:** 20px Outfit, `#8A96AA`, "AI, Automation, and Software Development Solutions" - **Primary CTA:** `Get Started` button, `#00F0FF` background, black text, soft cyan glow shadow - **Secondary CTA:** `View Services` button, transparent with 1px `#8A96AA` border, white text - **Background Glow:** Massive, subtle radial gradient in center-top, `#00F0FF` at 15% opacity fading to transparent **States:** - **Empty:** N/A - **Loading:** Entire screen enters via soft opacity fade-in (0.8s), CTA buttons slide up 20px. - **Error:** N/A **Components:** - **Hero Wrapper:** 100vh height, flex column, center aligned - **Nav Bar:** Fixed top, `backdrop-filter: blur(16px)`, height 72px **Interactions:** - **Hover Primary CTA:** Glow shadow intensifies to 30% opacity, scale to 1.02 - **Hover Secondary CTA:** Border changes to `#00F0FF`, text changes to `#00F0FF` **Responsive:** - **Desktop:** 1200px max-width container - **Tablet:** 48px headline, stacking CTAs - **Mobile:** 36px headline, hiding complex background glows to save performance ### Services View **Purpose:** Detail technical offerings without overwhelming the user. **Layout:** 2x2 grid of floating glass cards over a dark void with emerald accent glows. **Key Elements:** - **Section Title:** 40px Space Grotesk, "Core Capabilities" - **AI Card:** Glassmorphic panel, "AI & Deep Learning Solutions" title, predictive modeling icon - **Software Card:** Glassmorphic panel, "Custom Software Development" title, code bracket icon - **Automation Card:** Glassmorphic panel, "Intelligent Automation" title, workflow node icon - **Cloud Card:** Glassmorphic panel, "Cloud & DevOps Infrastructure" title, server stack icon **States:** - **Empty:** N/A - **Loading:** Cards stagger-fade in on scroll (IntersectionObserver) - **Error:** N/A **Components:** - **Service Card:** 320px width, 240px height, `rgba(17, 24, 39, 0.6)` background, 1px white/10 border **Interactions:** - **Hover Card:** Border illuminates `#00F0FF`, internal gradient shifts subtly, 4px upward translation **Responsive:** - **Desktop:** 2x2 grid, 40px gap - **Tablet:** 2x2 grid, 20px gap - **Mobile:** 1x4 stack ### Portfolio & About View **Purpose:** Prove competence through real projects and company mission. **Layout:** Left-aligned text for About, followed by a horizontal scrolling carousel of project cards. **Key Elements:** - **About Section:** 32px heading "AI-First. Future-Ready.", paragraph explaining mission - **Project 1 (CRM):** "CRM Dashboard", screenshot inset, `#00F0FF` tag "React/Node" - **Project 2 (Chatbot):** "AI Chatbot System", screenshot inset, `#00F0FF` tag "OpenAI/Python" - **Project 3 (Website):** "Business Website", screenshot inset, `#00F0FF` tag "Next.js/Tailwind" **States:** - **Empty:** "Check back soon for recent deployments" - **Loading:** Skeleton pulse for image insets (`#111827` to `#1F2937`) - **Error:** Fallback gradient if project image fails to load **Components:** - **Project Card:** 400px width, 16:9 image ratio, text block below, glass container **Interactions:** - **Hover Project:** Image scales up 1.05x inside hidden overflow container - **Click Tag:** Filters view (future implementation) **Responsive:** - **Desktop:** Horizontal overflow track - **Tablet:** Horizontal overflow track - **Mobile:** Vertical stack ### Contact & Footer View **Purpose:** Convert interest into direct communication. **Layout:** Split layout: Left side contact details/WhatsApp, Right side form. Footer spans bottom. **Key Elements:** - **Email Form:** Name, Email, Project Details fields, minimal lines (bottom border only inputs) - **WhatsApp CTA:** Emerald `#00FF9D` button, "Chat on WhatsApp", WhatsApp icon - **Direct Details:** `hello@aitdl.network`, simple monospace layout - **Footer Branding:** "AITDL Network", "Designed & Architected by Jawahar R Mallah", Social icons (X, LinkedIn, GitHub) **States:** - **Empty:** Form fields blank - **Loading:** "Sending transmission..." on submit button - **Error:** Red `#FF3366` bottom border on invalid fields **Components:** - **Input Field:** Transparent background, 1px `#8A96AA` bottom border, 16px Outfit - **Footer:** 80px height, 1px top border `#111827`, muted text **Interactions:** - **Focus Input:** Bottom border glows `#00F0FF`, label floats up - **Submit Form:** Button morphs to a checkmark, `#00FF9D` success glow **Responsive:** - **Desktop:** 50/50 split for contact, horizontal footer - **Tablet:** Stacked contact, horizontal footer - **Mobile:** Stacked contact, stacked centered footer

Build Guide **Stack:** HTML + Tailwind CSS v3, Alpine.js (for minimal interaction), Google Fonts (Space Grotesk, Outfit) **Build Order:** 1. **Design System & Base Shell:** Establish the `#070B14` background, global fonts, glassmorphism utility classes (`bg-gray-900/60 backdrop-blur-md border border-white/10`). 2. **Home View (Hero):** Critical for establishing the bioluminescent aesthetic. Dial in the radial gradients for the background glows before building the cards. 3. **Services View:** Reusable glass cards set the standard for the rest of the application's components. 4. **Portfolio & About:** Layout the horizontal scrolling section, utilizing standard Tailwind aspects for image containers. 5. **Contact & Footer:** Form validation states, custom input styling, and finalizing the layout flow.
